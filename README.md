# Crypto Trading Platform

AI-Powered crypto trading platform built with SprintBoot and React

## Backend

- SprintBoot
- SprintSecurity
- JWT
- Java Mailsender
- MySQL DB

## Frontend

- React
- Tailwind.css
- Redux (State management)
- Shadcn UI Components

## External APIs

- Gemini API -> AI Chatbot
- CoinGecko API -> Real-time cryptocurrency prices
